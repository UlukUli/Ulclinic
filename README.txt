Проект: Ulclinic — Консольная информационная система клиники
Автор: Samaganov Uluk COMSE24

Описание:
Программа моделирует работу клиники с ролями: врач, медсестра, главврач и пациент. Данные хранятся в базе SQLite.

Как запустить:
1. Открыть проект в IntelliJ IDEA
2. Перейти к файлу Main.java
3. Нажать ▶ (Shift + F10)
4. Войти, используя логин/пароль (см. ниже)

Вход:
- Врач: doc1 / 1234
- Медсестра: nurse1 / 1234
- Главврач: chief1 / 1234
- Пациенты: patient1 / 1234, akzhol / 1234

Что реализовано:

1. Авторизация:
- По логину и паролю, определение роли

2. Меню по ролям:
- Врач:
- Список пациентов
- Добавление диагноза из файла
- Назначение поручений медсестре
- Поиск пациентов
- Медсестра:
- Список процедур
- Выполнение поручений
- Главврач:
- Сотрудники, зарплаты
- Количество пациентов
- Пациент:
- История болезни
- Расписание врачей (из файла schedule.txt)
- Дни лечения

3. База данных:
- SQLite (clinic.db), создаётся автоматически
- Таблицы: users, patients, staff, tasks, procedures, medical_history

4. Вспомогательные файлы:
- diagnos.txt — список диагнозов с лечением
- schedule.txt — расписание приёма врачей

5. Очистка данных (для отладки):
- Методы deleteAllTasks(), debugShowAllTasks() и др.
